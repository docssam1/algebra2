1. 원본 `index.html`은 직접 대량 재작성하지 않는다.
2. 단계별 패치 전후로 변경 범위를 확인한다.
3. 백엔드, Cloud Run, API Key, 부모 발송, DB 파일은 건드리지 않는다.
4. 한글 인코딩은 UTF-8을 유지한다.
5. “무연근” 표현은 사용하지 않는다.
